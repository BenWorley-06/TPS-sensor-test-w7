Libraries Used:\
[Pressure Sensor (HX711) Library](https://github.com/RobTillaart/HX711/blob/master/examples/HX_plotter/HX_plotter.ino)\
[IMU Library](https://github.com/sparkfun/SparkFun_ICM-20948_ArduinoLibrary)\
\
When using pressure sensor always call sensor_inst.set_offset(offset value) and sensor_inst.set_scale(scale factor value)!/
/
Instructions to get started:\
Run HX calibration for the pressure sensor to get offset and scale factor values.\
OFFSET: 58269\
SCALE: -84.412163\
\
Configure IO pins\
Run IMU starter code and pressure sensor test to read values from sensors.
